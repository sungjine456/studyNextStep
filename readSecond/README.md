## 2독 진행
----

### 3장
```
	1. index.html 응답하기
		1) index.html로 페이지 이동
		2) url을 구하는 private 메소드 작성 후 reflection을 사용하여 테스트
		3) get 방식으로 회원가입
		4) post 방식으로 수정하여 회원가입
		5) 회원가입 후 index.html로 이동
		6) 로그인 후 쿠키 값 저장
		7) 로그인이 되었다면 사용자 목록 출력
		8) css 파일 지원
```
